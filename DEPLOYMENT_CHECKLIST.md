# Deployment Checklist

## Before Deployment

### Environment Variables
- [ ] Set NEXT_PUBLIC_SITE_URL to production URL
- [ ] Add Google AdSense Publisher ID
- [ ] Add Google Analytics Measurement ID
- [ ] Configure email service for contact form

### Content Review
- [ ] All legal pages completed (Privacy, Terms, Disclaimer)
- [ ] About page with company information
- [ ] Contact page functional
- [ ] At least 5-10 blog posts published
- [ ] All tool pages tested and working

### SEO Optimization
- [ ] Meta titles and descriptions on all pages
- [ ] Open Graph images created (1200x630)
- [ ] Favicon added
- [ ] Sitemap.xml accessible
- [ ] Robots.txt configured
- [ ] Schema markup implemented

### AdSense Requirements
- [ ] Privacy Policy published and linked
- [ ] Sufficient original content (10+ pages)
- [ ] Contact information visible
- [ ] Clear navigation
- [ ] Mobile responsive design
- [ ] Fast loading speed
- [ ] No prohibited content

### Testing
- [ ] Test all tools functionality
- [ ] Check mobile responsiveness
- [ ] Test contact form
- [ ] Verify all links work
- [ ] Check browser compatibility
- [ ] Test page load speed
- [ ] Verify analytics tracking

### Performance
- [ ] Images optimized
- [ ] CSS/JS minified
- [ ] Caching configured
- [ ] CDN setup (if applicable)

## After Deployment

- [ ] Submit sitemap to Google Search Console
- [ ] Verify Google Analytics is tracking
- [ ] Apply for Google AdSense approval
- [ ] Test production site thoroughly
- [ ] Set up monitoring/uptime tracking
- [ ] Create backups

## AdSense Application

1. Ensure domain is at least 6 months old (if new)
2. Have 15-20 pages of quality content
3. Submit application through adsense.google.com
4. Wait 2-4 weeks for review
5. Add ad code only after approval
